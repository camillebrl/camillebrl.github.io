Résumé du papier [Collaborative Large Language Model for Recommender Systems](http://arxiv.org/pdf/2311.01343v2).

Ce papier propose un nouveau modèle de recommandation appelé **CLLM4Rec** qui combine les avantages des modèles de langage à grande échelle (LLM) et des systèmes de recommandation basés sur les ID. En exploitant le pouvoir du traitement du langage naturel, CLLM4Rec est conçu pour surmonter le défi de la "lacune sémantique" entre les systèmes de recommandation (RS) et le langage humain.

LLM se réfère aux modèles de langage pré-entraînés qui ont une capacité de raisonnement logique et démontrent une compréhension sans précédent du langage naturel. Ils sont entraînés sur de grands corpus de données textuelles, tels que des articles Wikipedia, des articles de blog, des journaux, etc. Malgré leurs succès dans diverses tâches de traitement du langage naturel, leur application aux systèmes de recommandation a été limitée. Les systèmes de recommandation basés sur les LLM souffrent d’un certain nombre de défis, notamment l’inefficacité, l’instabilité et la dilution du vocabulaire.

CLLM4Rec s’attaque à ces défis en combinant les approches traditionnelles des systèmes de recommandation basés sur les ID (où chaque utilisateur et objet est représenté par un ID unique) et les LLM. Pour cela, il introduit des tokens ID utilisateur/objet dans le vocabulaire LLM, ce qui lui permet de capturer de manière fiable les sémantiques collaboratives et de contenu des utilisateurs et des objets.

Le modèle est entraîné en deux étapes. La première étape, appelée "pretraining stage", consiste à améliorer les tokens utilisateur/objet à l'aide de modèles d'apprentissage automatique régularisés mutuellement. Durant celle-ci, une nouvelle stratégie de "prompting" est proposée pour modéliser les documents avec des tokens hétérogènes.

La seconde étape, appelée "finetuning stage", ajuste le modèle appris dans la première étape pour prédire les éléments de recommandation basés sur des invites históriques d'interaction des utilisateurs. Les auteurs démontrent que cette "stratégie de finetuning" génère des recommandations de plusieurs éléments de manière efficace.

Les auteurs testent CLLM4Rec sur quatre jeux de données publics et un jeu de données de LinkedIn. Les résultats montrent que CLLM4Rec surpasse les approches existantes en matière de systèmes de recommandation basés sur les ID et les LLM. Il atteint également un niveau de performance pour la recommandation d'éléments qui est compétitif avec les approches basées sur l'apprentissage en profondeur.

En conclusion, CLLM4Rec représente une avancée majeure dans le développement de systèmes de recommandation en combinant avec succès les avantages du paradigme ID-based avec l'approche basée sur les LLM.
