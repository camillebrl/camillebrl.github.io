Résumé du papier [Towards Automated Negative Sampling in Implicit Recommendation](http://arxiv.org/pdf/2311.03526v1).

Dans le domaine des systèmes de recommandation, l'échantillonnage négatif est un mécanisme essentiel pour former des modèles efficaces. Cela consiste à sélectionner des exemples négatifs (c'est-à-dire des instances où un utilisateur n'a pas interagi avec un élément) pour compléter les exemples positifs dans l'ensemble de données d'entraînement. Cependant, le choix des échantillons négatifs peut être un défi et peut avoir un impact significatif sur les performances du modèle de recommandation. Ce document propose une nouvelle méthode pour automatiser le processus d'échantillonnage négatif. 

Les chercheurs ont utilisé diverses méthodes pour choisir des instances négatives difficiles, y compris, mais sans s'y limiter, le modèle de recommandation lui-même, le réseau antagoniste génératif et le modèle de graphe. Toutes ces méthodes visent à identifier les instances qui défient la capacité discriminatoire du modèle de recommandation. 

Cependant, ces approches négligent souvent le lien entre le modèle de recommandation, les ensembles de données implicites et les méthodes d'échantillonnage négatif. Les chercheurs remettent en question cette idée car elle contredit la croyance commune dans les tâches d'apprentissage automatique standard, comme la vision par ordinateur et le traitement du langage naturel, où l'architecture du modèle doit être typiquement sélectionnée pour s'adapter à l'ensemble de données cible afin d'obtenir de bonnes performances. Contrairement à ces tâches, la recommandation implicite ne contient que les instances positives de l'ensemble de données d'entraînement, tandis que l'échantillonneur négatif génère les instances négatives. 

Les chercheurs proposent une nouvelle méthode nommée AutoSample qui sélectionne automatiquement et de manière adaptative les stratégies d'échantillonnage pour différents modèles et ensembles de données. Un algorithme de recherche basé sur le gradient est proposé pour explorer efficacement l'espace de recherche. De plus, inspirés par l'apprentissage par curriculum, les auteurs proposent un schéma de réentrainement spécialisé pour faciliter les paramètres du modèle pendant la phase de recherche, ce qui conduit à une amélioration des performances.

Le papier introduit également un nouveau problème, appelé "échantillonnage négatif automatisé", qui consiste à formuler le choix des échantillonneurs négatifs comme un problème de recherche d'architecture neuronale. La méthode propose de reformuler le problème comme une somme pondérée de plusieurs pertes générées par différents échantillonneurs, ce qui permet une formation de bout en bout. 

Les expériences effectuées sur quatre ensembles de données publics et trois modèles de base montrent que AutoSample est capable de surpasser les autres échantillonneurs négatifs dans la plupart des cas. De plus, l'étude de cas montre que AutoSample présente un comportement d'apprentissage par curriculum, en ajustant progressivement les instances négatives pour faciliter un meilleur apprentissage.
