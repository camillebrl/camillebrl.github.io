Dans leur article, les auteurs discutent de l'application des modèles de langue pré-entraînés (MLPs) pour effectuer des prédictions sans être informés des relations dans un graphe de connaissances temporel (GTK). Les méthodes traditionnelles d'apprentissage pour les GTK sont limitées car elles ne peuvent pas gérer les relations non observées, également appelées relations "zéro-shot". Cependant, avec l'arrivée de nouveaux modèles linguistiques pré-entraînés à grande échelle (LLM), ces modèles peuvent être utilisés comme de puissantes bases de connaissances sémantiques. 

Les chercheurs proposent une nouvelle approche qui consiste à enrichir la description des relations dans les jeux de données GTK et à générer des représentations de ces relations en utilisant un LLM. Pour ce faire, ils utilisent le texte de description des relations fourni dans les jeux de données GTK et en génèrent une description enrichie (ERD) qui contient plus de détails pour chaque relation du graphe de connaissances (KG) à l'aide de LLM. 

Ensuite, ils génèrent les représentations des relations en utilisant un autre LLM, le T5-11B. Ils introduisent les ERDs dans l'encodeur de T5 et transforment sa sortie en représentations de relations pour les modèles GTKF. Ils conçoivent également un apprenant d'historique des relations (RHL) pour saisir les modèles historiques des relations, en tirant parti des représentations de relations améliorées par les LLM pour un meilleur raisonnement sur les relations zéro-shot. 

L'auteur précise également que cette nouvelle approche, appelée zrLLM, est compatible avec diverses méthodes de prévision pour les GTK. C'est pourquoi ils réalisent des expériences sur sept modèles récents basés sur l'apprentissage en profondeur pour les GTK et les évaluent sur trois nouveaux jeux de données conçus spécifiquement pour l'étude de l'apprentissage relationnel zéro-shot dans les GTK.

Dans leur étude comparative, ils montrent que zrLLM aide à améliorer considérablement les capacités de tous les modèles GTK considérés pour prévoir les faits contenant des relations zéro-shot non vues. Cela montre que zrLLM est efficace et fortement adaptable. En résumé, ils prouvent que les approches basées sur les LLM peuvent augmenter les performances de prédiction des relations dans les GTK, y compris pour les relations jamais vues auparavant.
