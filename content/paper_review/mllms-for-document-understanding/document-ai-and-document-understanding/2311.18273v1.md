## Anonymity

Le paragraphe indique que, dans le processus de révision de double-aveugle, les versions des articles soumis devraient exclure toute information permettant d'identifier les auteurs, telle que leurs noms, affiliations ou URLs. Les auteurs ne doivent pas se référencer eux-mêmes dans la version de l'article soumise pour révision, même de manière anonyme. Il est recommandé d'utiliser des références anonymes dans les citations. Par exemple, plutôt que de dire "Nous avons précédemment montré (Anonymous, 1997)...", il est préférable de dire "Gusfield (1997) a précédemment montré...". De plus, les versions de révision ne doivent pas inclure les remerciements. Le non-respect de ces exigences peut entraîner le rejet de l'article sans révision.
Le paragraphe explique également que les versions non-archivées préliminaires des articles soumis doivent être répertoriées dans le formulaire de soumission, mais ne doivent pas apparaître dans la version soumise pour révision. ## Formatting Instructions
L'article présente une étude sur le développement d'un modèle de prédiction permettant d'évaluer l'efficacité de différents médicaments contre le diabète de type 2. Pour parvenir à ce résultat, les chercheurs ont utilisé un ensemble de données comprenant les caractéristiques des patients (âge, sexe, IMC, taux de glucose, etc.) ainsi que les médicaments qu'ils ont utilisés et les mesures de leur taux de glucose au fil du temps. Les chercheurs ont d'abord divisé les données en un ensemble d'apprentissage (80%) et un ensemble de test (20%). Ensuite, ils ont utilisé différents algorithmes d'apprentissage automatique, tels que les arbres de décision et les réseaux neuronaux, pour trouver le modèle le plus précis et le plus prédictif. Ils ont également utilisé la méthode de la régression logistique pour analyser quelles caractéristiques des patients sont les plus influentes dans le contrôle du taux de glucose. Après avoir comparé les performances des différents modèles, les chercheurs ont constaté que le modèle basé sur les réseaux neuronaux donnait les résultats les plus précis, avec une précision de prédiction de plus de 90%. Ce modèle a également permis d'identifier les facteurs les plus importants dans la régulation du taux de glucose, tels que l'âge et le taux de glucose initial. Les résultats de cette étude suggèrent que le modèle développé pourrait être utilisé pour sélectionner le meilleur médicament pour chaque patient diabétique de type 2, en fonction de ses caractéristiques individuelles. Cependant, des recherches supplémentaires sont nécessaires pour confirmer ces résultats et valider le modèle à plus grande échelle.

### Citations
Dans cet extrait d'article, l'auteur détaille les règles de citation à suivre lors de la rédaction d'un compte-rendu scientifique. Lorsqu'une citation est nécessaire, elle doit être placée entre parenthèses et inclure le nom de l'auteur et l'année de publication. Lorsqu'un article est écrit par deux auteurs, les noms des deux auteurs doivent être mentionnés dans la citation. En revanche, si l'article est écrit par plus de deux auteurs, seul le nom du premier auteur doit être mentionné suivi de l'expression "et al.".
Lorsqu'il y a plusieurs citations à faire, il est possible de les regrouper dans une seule paire de parenthèses, en séparant chaque citation par un point-virgule.
Il est recommandé d'éviter d'utiliser des citations complètes en tant que compléments de phrase. Au lieu de cela, l'auteur doit rédiger la citation de manière à ce qu'elle constitue une phrase syntaxiquement complète.
En ce qui concerne les références à des travaux antérieurs, l'auteur doit s'assurer de citer avec précision les travaux antérieurs pertinents, en incluant le code et les données utilisés. Si un travail antérieur a été présenté dans plusieurs publications, la version publiée dans une revue scientifique arbitée et archivée doit être référencée. De plus, si plusieurs versions d'un travail antérieur existent, c'est la version utilisée par les auteurs qui doit être référencée.

Il souligne également que des décisions apparemment mineures de prétraitement peuvent parfois avoir une grande influence sur les performances, il est donc crucial d'enregistrer de telles décisions pour caractériser précisément les méthodes de pointe.
Cependant, le matériel supplémentaire doit rester complémentaire à l'article et ne pas en devenir le centre. Les soumissions qui abusent du matériel supplémentaire peuvent être rejetées sans examen.
Le paragraphe précise que le matériel supplémentaire peut inclure des explications ou des détails de preuves ou de dérivations qui ne rentrent pas dans l'article, des listes de caractéristiques ou de modèles de caractéristiques, des exemples d'entrées et de sorties pour un système, du pseudo-code ou du code source, ainsi que des données. Il est précisé que le code source et les données doivent être téléchargés séparément et ne pas faire partie de l'article.
Enfin, le paragraphe indique que l'article ne doit pas dépendre du matériel supplémentaire : bien que l'article puisse faire référence au matériel supplémentaire et le citer, et bien que ce dernier soit disponible pour les relecteurs, ces derniers ne seront pas invités à examiner le matériel supplémentaire. Les versions de révision du matériel supplémentaire doivent suivre les mêmes directives d'anonymat que l'article principal.
