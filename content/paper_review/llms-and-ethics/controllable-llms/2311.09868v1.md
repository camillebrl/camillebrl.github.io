Résumé du papier "Prompting the Coding Ability of Large Language Models with the Interactive Chain of Repairing".
---

Le document explore l'interaction interactive de modèles de langage à grande échelle pour résoudre des problèmes de codage. Il présente comme hypothèse que des chaînes d'interactions peuvent améliorer la capacité de ces modèles à générer du code. Il introduit une nouvelle méthodologie appelée "Interactive Chain of Repairing" (ICR) qui promet d'améliorer les performances des modèles de langage dans la résolution de problèmes de codage. 

L'introduction décrit le contexte général de l'exploration des capacités de codage des modèles de langage à grande échelle. La section "Travaux connexes" mentionne d'autres modèles de langage capables de produire du code informatique, tels que Codex, Text-DaVinci et ChatGPT. 

La méthodologie ICR est ensuite présentée de manière détaillée, expliquant comment elle repose sur l'utilisation de chaînes d'interactions pour guider les modèles de langage dans la production de code. Les modèles de langage sont initiés avec un programme de code incorrect, puis ils sont invités à le corriger au fur et à mesure qu'ils interagissent avec l’ICR. Cette méthode diffère des approches traditionnelles où le modèle de langage tente de générer un code correct à la première tentative. 

La méthodologie expérimentale comprend également le choix de l'ensemble de données, les mesures d'évaluation, les modèles de base et les détails de mise en œuvre. L'ensemble de données contient des extraits de code Python provenant de différentes sources, et les mesures d'évaluation comprennent des mesures telles que la précision du code généré et le nombre moyen d'interactions nécessaires pour produire un code correct.

Des résultats d’évaluation sont présentés pour illustrer les performances de l'ICR. Un certain nombre de modèles de langage, y compris certains de ceux mentionnés dans la section "Travaux connexes", sont utilisés en tant que modèles de base auxquels les performances de l'ICR sont comparées. Les résultats de l'évaluation révèlent que l'ICR est capable d'améliorer de manière significative les performances de ces modèles de langage. 

Enfin, le document conclut sur le potentiel de l'ICR pour améliorer les capacités de codage des modèles de langage à grande échelle. Il souligne également la nécessité d'autres recherches pour explorer d'autres aspects de cette approche, comme l'efficacité de différents types d'interactions dans le processus de réparation de code.
