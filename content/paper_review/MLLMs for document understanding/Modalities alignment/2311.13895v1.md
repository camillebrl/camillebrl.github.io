Ce compte-rendu résume le papier "Query by Activity Video in the Wild", qui se concentre sur la recherche d'activités à partir d'une vidéo quand les exemples d'activités sont déséquilibrés. L'hypothèse commune dans la littérature actuelle est que toutes les activités ont suffisamment d'exemples étiquetés pour apprendre un plongement dans l'espace d'activités. Cependant, ce n'est pas le cas dans la pratique, car seules certaines activités ont beaucoup d'exemples, tandis que d'autres ne sont décrites que par quelques-uns. Pour surmonter ce problème, les auteurs proposent un réseau d'incrustation visuelle-sémantique qui gère explicitement le scénario déséquilibré pour la recherche d'activités. 

Le réseau contient deux modules novateurs. Le module d'alignement visuel effectue un alignement global entre la vidéo d'entrée et les représentations de banques visuelles de taille fixe pour toutes les activités. Le module sémantique effectue un alignement entre la vidéo d'entrée et les représentations sémantiques d'activités de taille fixe. Ces modules visent explicitement le problème du déséquilibre dans l'ensemble de données vidéo. 

Les expériences sur un nouveau benchmark de recherche d'activités déséquilibré montrent l'efficacité de l'approche pour tous les types d'activités. Il est à noter que cette approche ne se limite pas à la recherche d'activités par vidéo, elle permet aussi de récupérer des clips vidéo et des moments vidéo à partir de requêtes vidéo dans des scénarios déséquilibrés.

Les auteurs ont effectué une série d'analyses pour comprendre le problème et leur solution. Ils ont constaté que même lorsqu'un seul échantillon de classe est fourni, leur méthode peut capturer des connaissances à partir d'une supervision limitée. De plus, ils ont constaté que dès que le nombre d'exemples pour les classes nouvelles augmente, l'écart avec la base de référence augmente également. 

En conclusion, les auteurs ont proposé une nouvelle tâche de récupération de vidéos par activité dans des environnements non contrôlés, et ont souligné l'importance de traiter les données déséquilibrées lors de la récupération d'activités à partir d'une requête vidéo.
