Résumé du papier [Retrieval-Augmented Code Generation for Universal Information Extraction](http://arxiv.org/pdf/2311.02962v1).

Code4UIE est un nouveau cadre proposé pour l'extraction d'informations universelles, qui vise à extraire des connaissances structurées à partir de textes en langage naturel. Les défis posés à cette extractions comprennent des schémas spécifiques à la tâche et des expressions de texte complexes. Pour résoudre ces défis, Code4UIE propose de transformer l'ensemble des tâches d'extraction d'informations en une tâche unifiée de génération de code en utilisant des classes Python pour définir les schémas spécifiques à chaque tâche de manière universelle. Par conséquent, l'extraction de connaissances peut être réalisée en générant des codes qui instancient les classes Python prédéfinies avec les informations contenues dans les textes.

Pour générer ces codes de manière plus précise, Code4UIE adopte le mécanisme d'apprentissage en contexte pour instruire les grands modèles linguistiques (LLMs) à l'aide d'exemples. Afin d'obtenir des exemples appropriés pour différentes tâches, Code4UIE explore plusieurs stratégies de récupération d'exemples, qui peuvent récupérer des exemples sémantiquement similaires aux textes donnés.

Des expériences sur cinq tâches représentatives d'extraction d'informations dans neuf ensembles de données démontrent l'efficacité du cadre Code4UIE. En résumé, les auteurs ont proposé une méthode de représentation basée sur un schéma qui peut définir universellement divers schémas spécifiques à une tâche dans les tâches d'extraction d'informations en termes de classes Python. Ils ont transformé les tâches d'extraction d'informations en une tâche unifiée de génération de code basée sur la méthode de représentation du schéma et ont proposé un cadre de génération de code augmenté par la recherche basé sur les LLMs, Code4UIE, pour résoudre cette tâche de génération de code unifiée. Code4UIE est équipé de stratégies unifiées de récupération d'exemples pour aider les LLMs à mieux comprendre les expressions de texte complexes par apprentissage en contexte.

Les relations et les événements sont décomposés en différentes classes Python, avec des exemples définis pour guider le modèle à produire les codes appropriés. Les performances de Code4UIE sont comparées à celles des modèles superviseurs pour l'extraction d'informations sur différentes tâches, montrant qu'il améliore significativement les performances par rapport aux méthodes basées sur des modèles de langage volumineux existants pour toutes les tâches d'extraction d'informations.
