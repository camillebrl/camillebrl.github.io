Résumé du papier "When Large Language Models contradict humans? Large Language Models' Sycophantic Behaviour".

Le document fournit un ensemble d'instructions détaillé pour la soumission de documents pour les conférences ACL. Tous les auteurs sont tenus de respecter ces spécifications, qui comprennent des directives sur le style, la longueur du papier, l'anonymat, les multiples soumissions, les instructions de formatage, la mise en page, les polices, le titre et les auteurs, le résumé, le texte, les sections, les notes de bas de page, les figures et les tableaux, les liens hypertextes, le texte en langues autres que l'anglais, les citations, les remerciements, les références, les annexes et le matériel supplémentaire. 

Il est recommandé d'utiliser les fichiers de style fournis par ACL pour respecter ces exigences. Le document détaille la longueur autorisée pour les versions d'examen des documents longs et courts, ainsi que pour leurs versions finales. Les auteurs sont encouragés à soumettre des annexes et du matériel supplémentaire, qui ne sont pas tenus de respecter ces limites de pages. 

Le processus d'examen étant en double aveugle, les versions d'examen ne doivent pas inclure d'informations d'identification sur les auteurs. Les auteurs qui soumettent plusieurs documents à ACL doivent veiller à ce que leurs soumissions ne se chevauchent pas de manière significative. 

Le document explique comment le texte doit être formaté, avec des instructions détaillées sur le placement du texte, la taille et le style des polices, les sections, les notes de bas de page, les figures et les tableaux, les citations et les références. Il fournit également des conseils sur la citation d'articles dans différents types de publications, et sur l'inclusion de DOI ou de liens vers les pages de l'ACL Anthology lorsque c'est possible. 

Les auteurs sont également encouragés à inclure des annexes et du matériel supplémentaire, qui doivent suivre les mêmes directives d'anonymat que le reste du document. 

Enfin, le document remercie tous ceux qui ont contribué à la rédaction de ces instructions au fil des ans.
