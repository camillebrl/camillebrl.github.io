Résumé du papier "Techniques de classification de texte explicables dans l'examen de documents juridiques : localisation des rationales sans utiliser des extraits de texte de formation annotés par l'homme".

Ce papier propose deux méthodes simples de machine learning pour localiser les extraits de texte répondant à certains critères (alias rationales) dans des documents juridiques sans utiliser d'extrait de texte pour la formation annotée par l'humain.

En l'absence d'annotations humaines, la tâche de trouver les extraits de texte pertinents est complexe mais capitale pour expliquer les décisions de classification d'un modèle de machine learning. Les auteurs ont mis en place deux approches pour résoudre ce problème, en utilisant le machine learning.

La première méthode, appelée "Snippet Model Method", utilise un modèle de texte au niveau du document pour marquer tous les extraits de texte chevauchants dans tous les documents de la formation. Ensuite, un ensemble de snippets à fort score provenant de documents de formation réactifs est sélectionné comme snippets de formation réactifs. Ces snippets sont ensuite utilisés pour former un modèle de détection, qui est utilisé pour localiser des extraits de texte réactifs dans de nouveaux documents.

La deuxième méthode, appelée "Méthode du modèle de snippet itératif", utilise une approche itérative pour affiner le nombre de tokens dans les snippets de texte de formation. Elle commence par un modèle de document complet et l'applique pour identifier de grands snippets de formation réactifs. Lors de chaque itération, un nouveau modèle de snippet est formé en utilisant un nouvel ensemble de snippets de formation, et ce modèle est ensuite utilisé pour identifier un nouvel ensemble de snippets de formation, où la taille des tokens du snippet est réduite de moitié à chaque itération.

Les auteurs ont testé ces deux méthodes sur trois jeux de données provenant de situations juridiques réelles et ont constaté qu'elles surpassaient la méthode de classification de documents au niveau de la formation dans l'identification de snippets réactifs dans les documents réactifs. Ces résultats suggèrent qu'il est possible d'automatiser l'identification réussie du texte d'entraînement pour améliorer la précision des prédictions dans l'examen de documents juridiques et donc aider à réduire le coût global de l'examen.

Globalement, ce travail démontre qu'il est possible de construire des modèles de machine learning capables d'identifier automatiquement des rationales sans utiliser des extraits de texte annotés pour la formation. Les auteurs prévoient de mener des expériences supplémentaires avec plus de jeux de données et d'explorer d'autres technologies de machine learning avancées pour améliorer ces approches.
