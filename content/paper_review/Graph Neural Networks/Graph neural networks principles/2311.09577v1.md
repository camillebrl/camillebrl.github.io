Le papier intitulé ["Group-Aware Interest Disentangled Dual-Training for Personalized Recommendation"](http://arxiv.org/pdf/2311.09577v1) de Xiaolong Liu et al. se concentre sur l'amélioration des systèmes de recommandation personnalisés en utilisant les informations de groupe des utilisateurs. Les auteurs ont développé un modèle appelé IGRec, qui désintègre les intérêts des utilisateurs en différentes représentations d'intérêt et utilise ces informations pour améliorer les recommandations.

Dans le modèle IGRec, l'information de groupe d'un utilisateur est utilisée pour désintégrer les intérêts de l'utilisateur en différentes représentations grâce à une unité d'auto-contrôle. Chaque unité d'auto-contrôle est formée pour extraire les différents intérêts des utilisateurs à partir de leur représentation initiale. Cela permet au modèle de reconnaître que les utilisateurs peuvent avoir des intérêts multiples et divers qui peuvent être révélés par leur participation à différents groupes.

Le modèle génère ensuite une représentation de groupe basée sur les intérêts en utilisant une agrégation Gumbel-Softmax sur les intérêts des membres du groupe. Cette représentation est ensuite fusionnée avec la représentation de l'utilisateur pour créer une représentation améliorée de l'utilisateur qui est utilisée pour les prédictions de notation. Enfin, le modèle est formé en double pour les interactions entre les utilisateurs et les groupes.

Les auteurs ont testé le modèle IGRec sur trois ensembles de données publics et ont montré qu'il peut atténuer efficacement le problème de la rareté des données et améliorer les systèmes de recommandation en utilisant une représentation de groupe basée sur l'intérêt. Dans les expériences de recommandation de groupe, le modèle a également montré que la représentation basée sur l'intérêt du groupe est informative.

Cette recherche montre comment l'information de groupe peut être utilisée pour améliorer les recommandations personnalisées. L'approche peut être particulièrement utile pour les plateformes en ligne où les utilisateurs forment des groupes en fonction de leurs intérêts communs. Cependant, les auteurs notent également que l'efficacité du modèle dépend fortement de la disponibilité et de la qualité de l'information sur le groupe. Un futur travail pourrait inclure l'exploration de comment ces informations de groupe pourraient être recueillies de manière plus efficace et omplète.
